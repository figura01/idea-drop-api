# Idea Drop API

Développement de l'api backend Idea Drop avec node, express, mongoDB

# Technos

- Node
- Express
- MongoDB
- JWT
- Typescript

# Lancement

1. Clone du repos

```
git clone git@github.com:figura01/idea-drop-api.git
```

2. Installer les dépendences

```
npm i
```

3. Créer un .env a partir de l'exemple et remplacer les variables d'environnements

4. Lancer le serveur

```
npm run start
```
